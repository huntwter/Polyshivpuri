## Project Overview
Polyshivpuri is the official institute website developed for Government Polytechnic College Shivpuri.  
It provides centralized access to academic, administrative, and student-related information through a clean and responsive interface.

---

## Tech Stack
- HTML5  
- CSS3  
- JavaScript
